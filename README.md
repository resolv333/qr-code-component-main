### What I learned

I learned a lot while building this project. Such as how to look up info to get help when I got stuck. I mostly used google to help me, but also a little of w3schools.com. 

I also leaned how to got about centering things and postioning items inside of a div. First I had to figure out that the image and the text were going inside of a div so that the div could be white and the background of the site a slightly white-blue color. It made an interesting affect.

I liked doing this project and took me about a day to do.


### Continued development

I want to focus more on divs and even more so semantic html. I also want to learn more about classes and how to style each element as I want with css. Positioning things is still something that takes me a little while to figure out.
